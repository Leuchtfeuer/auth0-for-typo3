===========================
Version 13.0.3 - 2026/01/07
===========================

This is a Feature Release with security updates, new configuration for sudo mode bypass,
password change functionality for Auth0 users, TYPO3 version requirement update, global
TypoScript provision, and login logo bug fixes.

Download
========

Download this version from the `TYPO3 extension repository <https://extensions.typo3.org/extension/auth0/>`__ or from
`GitHub <https://github.com/Leuchtfeuer/auth0-for-typo3/releases/tag/v13.0.3>`__.
