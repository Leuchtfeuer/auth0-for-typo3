==========================
Version 5.1.1 - 2026/02/04
==========================

This release is a minor stability release which adds symfony/http-foundation:6.4 as a possible composer solution
to improve TYPO3 v12 support with PHP 8.1.

Download
========

Download this version from the `TYPO3 extension repository <https://extensions.typo3.org/extension/auth0/>`__ or from
`GitHub <https://github.com/Leuchtfeuer/auth0-for-typo3/releases/tag/v5.1.1>`__.
